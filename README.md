# 🛠 План розробки проєкту: [Lifestyle]
 
## 1. Концепція продукту
- **Тип додатку:** [ Веб]
- **Основна ідея:** [    Простий онлайн-щоденник, де користувачі можуть записувати свій день, думки та настрій без оцінювання іншими]
 
## 2. Основний функціонал (Features)
* Приймати текст від користувача (запис дня)
* Зберігати записи
* Переглядати записи за датами (календар)
* Обирати настрій дня
* Режим анонімного запису
* Нагадування про запис
* Редагування або видалення записів
 
## 3. Структура даних
Назва змінної | Тип даних | Для чого потрібна

|user_input | str | Текст запису користувача|

|date | date | Дата запису|

|mood | str | Настрій (радісний, сумний тощо)|

|is_private | bool | Чи є запис приватним|

|user_id | int | Ідентифікатор користувача|

|is_active | bool | Контроль роботи додатку|
